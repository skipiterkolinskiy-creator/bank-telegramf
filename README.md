# Z-Bank Telegram Bot

Полностью переписанный бот на Python 3.13 и aiogram 3.x.

## Запуск

1. Установите зависимости:

```bash
pip install -r requirements.txt
```

2. Создайте `.env` по примеру `.env.example`:

```env
BOT_TOKEN=ваш_токен
STAFF_GROUP_ID=-1003993331647
DATABASE_DIR=database
LEGACY_DATA_PATH=C:\Users\skipi\Downloads\data.json
```

На BotHost токен обычно лежит в системной переменной `API_TOKEN`. Код поддерживает оба варианта, но на хостинге приоритет у `API_TOKEN`, чтобы старый пользовательский `BOT_TOKEN` не перебивал правильный токен.

Для постоянной базы на BotHost можно не задавать `DATABASE_DIR`: код сам возьмет `DATA_DIR=/app/data`.

3. Запустите:

```bash
python bot.py
```

## Миграция

При первом запуске бот автоматически создаст JSON-базу и перенесет старый `data.json`, если `database/users.json` пустой.

Порядок поиска старой базы:

1. `LEGACY_DATA_PATH` из `.env`
2. `data.json` рядом с проектом
3. `database/data.json`
4. `C:\Users\skipi\Downloads\data.json`

## Админка

Команда `/panel` работает только в staff-чате.

По умолчанию staff-чат: `-1003993331647`.

Роли, владельцы и `admins.json` для доступа к панели больше не нужны. Любой участник staff-чата может открыть `/panel` и выполнять действия панели: поиск клиента, пополнение баланса, списание баланса, заморозка/разморозка аккаунта, отметка проверки, управление доступом, просмотр лицензий, инвентаря и логов.
